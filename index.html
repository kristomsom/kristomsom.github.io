Script de Regresión Lineal Múltiple en R

Este código realiza un análisis de regresión lineal múltiple sobre datos de abundancia de artrópodos en relación con variables ambientales.


# Cargar los datos
datos <- read.csv("C:/Users/Usuario/Documents/R studios/datos_regresion_lineal_multiple.csv")

# Subconjunto de variables predictoras para correlación
predictoras <- datos[, c("canopy_cover", "av_slope", "trees_over_3m", "av_litter_depth")]

# Correlación de Pearson
cor(predictoras, use = "complete.obs", method = "pearson")

# Crear modelos
m1 <- lm(Abundance ~ canopy_cover, data = datos)
m2 <- lm(Abundance ~ canopy_cover + av_slope, data = datos)
m3 <- lm(Abundance ~ canopy_cover + trees_over_3m, data = datos)
m4 <- lm(Abundance ~ canopy_cover + av_litter_depth, data = datos)
m5 <- lm(Abundance ~ canopy_cover + av_slope + trees_over_3m + av_litter_depth, data = datos)

# Evaluar resumen del mejor modelo
summary(m5)

# Visualizar relación simple
plot(datos$canopy_cover, datos$Abundance,
     main = "Relación entre canopy cover y riqueza de artrópodos",
     xlab = "Canopy Cover (%)", ylab = "Riqueza de Artrópodos")
abline(m1, col = "blue", lwd = 2)

# Mostrar 4 gráficos de diagnóstico en una cuadrícula 2x2
layout(matrix(c(1,2,3,4), 2, 2)) 
plot(m5)

# Instalar y cargar jtools
install.packages("jtools")  # Solo la primera vez
library(jtools)

# Gráfico tipo "effect_plot" de jtools para canopy_cover
effect_plot(m5, pred = canopy_cover, interval = TRUE, plot.points = TRUE,
            jitter = 0.05, x.label = "Cobertura de dosel (%)",
            y.label = "Riqueza de artrópodos",
            main.title = "Efecto de canopy_cover sobre la riqueza de artrópodos")

Interpretación del análisis:

Decidí hacer una regresión lineal múltiple para evaluar cuáles de estas variables influyen sobre la riqueza de artrópodos. Primero, evalué si las variables predictoras tenían correlaciones entre sí usando Pearson, luego ajusté varios modelos combinando las variables predictoras y los comparé usando AIC.

El modelo con el menor AIC fue el m5, que incluye las variables canopy_cover, av_slope, trees_over_3m y av_litter_depth. Este modelo explica aproximadamente el 52% de la varianza observada en la riqueza de artrópodos (R² ajustado = 0.52), y muestra que la variable canopy_cover tiene un efecto significativo (p = 0.0018). Los análisis de diagnóstico indican que se cumplen los supuestos de normalidad y homocedasticidad, sin evidencia fuerte de observaciones influyentes.

Ejemplo general para aplicar regresión múltiple


# 📦 Instalar e importar paquete para gráficos de regresión
install.packages("jtools")  # Ejecutar solo la primera vez
library(jtools)

# 📁 Cargar los datos
datos <- read.csv("RUTA/AL/ARCHIVO.csv")  # Reemplaza con la ruta real

# 🔍 Seleccionar variables predictoras numéricas/continuas
predictoras <- datos[, c("variable_numerica_1", "variable_numerica_2", 
                         "variable_numerica_3", "variable_numerica_4")]

# 📈 Calcular correlación entre las variables predictoras
cor(predictoras, use = "complete.obs", method = "pearson")

# 🧮 Crear varios modelos de regresión lineal múltiple
modelo1 <- lm(variable_dependiente ~ variable_numerica_1, data = datos)
modelo2 <- lm(variable_dependiente ~ variable_numerica_1 + variable_numerica_2, data = datos)
modelo3 <- lm(variable_dependiente ~ variable_numerica_1 + variable_numerica_3, data = datos)
modelo4 <- lm(variable_dependiente ~ variable_numerica_1 + variable_numerica_4, data = datos)
modelo5 <- lm(variable_dependiente ~ variable_numerica_1 + variable_numerica_2 + 
                               variable_numerica_3 + variable_numerica_4, data = datos)

# 📉 Comparar los modelos por AIC para seleccionar el mejor
AIC(modelo1, modelo2, modelo3, modelo4, modelo5)

# ✅ Ver el resumen del mejor modelo
summary(modelo5)

# 📊 Graficar relación simple
plot(datos$variable_numerica_1, datos$variable_dependiente,
     main = "Relación entre variable predictora y variable dependiente",
     xlab = "Variable numérica", ylab = "Variable dependiente")
abline(modelo1, col = "blue", lwd = 2)

# 📋 Gráficos de diagnóstico
layout(matrix(c(1, 2, 3, 4), 2, 2))
plot(modelo5)

# 🎨 Gráfico del efecto parcial
effect_plot(modelo5, pred = variable_numerica_1, interval = TRUE, plot.points = TRUE,
            jitter = 0.05, x.label = "Variable predictora",
            y.label = "Variable dependiente",
            main.title = "Efecto de la variable sobre la variable dependiente")

🔁 1. El mejor modelo puede cambiar: Depende del ecosistema o contexto.

📉 2. El R² ajustado puede variar: Puede subir o bajar según la calidad de las variables predictoras.

⭐ 3. Las variables significativas cambian: Dependen de tamaño de muestra, colinealidad, etc.

⚠️ 4. Se pueden violar supuestos: Normalidad, homocedasticidad y outliers deben verificarse en cada caso.

💡 5. El comportamiento de los predictores puede no ser lineal: Considerar términos cuadráticos o modelos no lineales.
